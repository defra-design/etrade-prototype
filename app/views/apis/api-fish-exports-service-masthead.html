This API lets exporters create catch certificates for fish and shellfish using Defra’s Fish Exports Service.

Introduction

You must create a catch certificate to export a consignment of fish that has been landed into the UK by a UK flagged catching vessel. A catch certificate shows that you’ve caught your fish legally. Catch certificates are required for movements of fish from:

  • Great Britain to Northern Ireland
  • Great Britain to the EU the UK to the rest of the world (excluding movements from Northern Ireland to the EU)

Find out more about exporting fish

Benefits of using this API

This API enables system-to-system calls between your own software and the Fish Exports Service. This makes it easier to apply for a catch certificate and reduces the admin burden on exporters. It also reduces the risk of errors and data entry duplication.

What it can do

When the API is ready, you'll be able to use it to create catch certificates for fish and shellfish.

This API can:

  • retrieve lists of valid catch areas and departure countries
  • check a vessel is eligible to create a catch certificate
  • search valid product species, preservation and presentation methods for creating catch certificates

Who can use it

Importers, traders and agencies will find this API useful.

You’re welcome to try the current sandbox service. Only a small number of invited users can currently progress to full production integration. If you don't have an invite, you’re welcome to integrate with our sandbox environment in order to test your integration with Defra APIs ready for full integration when the service is ready. How to export using this API

Before you login

Make sure you have read the full guidance on exporting fish before you begin using this API.

The endpoint in this API is user-restricted. Your users must register as an individual before they can submit applications using the API.

Once you're fully integrated with the API you'll follow these steps to export goods:

  1. Complete and submit your application using your integrated systems. You may also need to create an export health certificate, which you can do using a different Defra API. You may also need a storage document and processing statement.
  2. You can download and print your catch certificate from your integrated systems.
  3. The physical catch certificate and other documentation should accompany the consignment as it leaves the UK.
