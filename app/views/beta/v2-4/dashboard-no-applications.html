We’ve made some changes to the service

To make sure all export data is consistent with the EU’s trade and control expert system (TRACES), we’ve made some changes to how you apply for an export health certificate (EHC).

If you apply for an EHC through the new service, your:

  • certifiers and importers will no longer need to use paper copies of EHCs
  • commodity data will be validated by TRACES before you submit it to the certifier

You can still use the previous EHC application service if you want to:

  • view an application that you started before 1 September 2022
  • apply for an EHC for exports to countries outside the EU or Northern Ireland
