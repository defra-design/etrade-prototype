Getting production credentials

If your appliucation subscribes to an API only available in the sandbox, you cannot request production credentials. Review the specific API documentation to check if an API is available in production.

You will need to test your application before requesting production credentials.

You do not need production credentials for APIs with only open access endpoints or:

  • APIs for testing your service
  • Hello world API
What the sandbox lets you do
  • create your sandbox application
  • subscribe it to the APIs you want to integrate with
  • get access to its testing credentials
  • use its testing credentials for integration testing
