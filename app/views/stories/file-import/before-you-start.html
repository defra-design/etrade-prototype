Before you start

To make sure all export data is consistent with the EU’s trade and control expert system (TRACES), we have made some changes to how you enter commodity data.

If data is consistent between exporters in Great Britain and importers in Northern Ireland and the EU, exports are more likely to pass through border control posts and reach their destinations.

You’ll need to enter data for the export’s commodity group, weight and quantity before submitting the application to the certifier.

If you want to upload commodity data through a spreadsheet

You must use the updated spreadsheet template, as it follows EU TRACES data requirements.

You can enter up to 100 rows of data for up to 15 different commodities. If you want to enter more commodity data, you can upload multiple spreadsheets.
