You can now submit a digital application to the EU and Northern Ireland

For applications to the EU and Northern Ireland, you can create a paperless EHC with data that aligns to the EU's legislative requirements (CHED) and their IT system (TRACES).

How it works

  1. You

    1. Create application
    A digital application is similar to paper but some data you must select, such as commodity codes and premises.

  2. Certifier

    2. Review and submit
    The data is reviewed, digitally approved and validated against TRACES, replacing a printed certificate.

  3. Importer

    3. Create CHED
    All data is sent to TRACES as an EU Import Notice and is copied into a CHED, and the two are linked.

  4. Transporter

    4. Give QR code to BCP
    You will give a QR code to the transporter, enabling the BCP to locate the linked Import Notice and CHED.

What has changed

  • No requirement to print and stamp EHCs
  • Data validation with the EU’s requirements prior to dispatch
  • Data matching between the EU Import Notice and CHED
  • Reduced burden on importer when creating the CHED
  • Data can be changed after the goods have been dispatched in the digital service, such as transport changes or BCP feedback (Cancel & reissue)

What remains the same

  • Certifiers will continue to check data and the goods
  • Data (and sometimes goods) are checked by BCPs (a separate process to data validation)
  • A previous digital application can be copied

Before you start

Allow adequate time for your first application.

Paper applications do not send data to TRACES. To make use of the data in your application, the person creating the CHED will need to find the EU import notice. This process is new. You must make the person that creates the CHED aware of the new process before you start. If you’re not sure who does this, contact your customer.

How the CHED is created when making a digital application

Once your Certifier has approved an application and the data is received in TRACES, you will be given an IMSOC (Import Model) reference which you must send to the person that creates the CHED.

These steps will be new to the person that creates the CHED:

  1. "Documents" menu
  2. "EU Import"
  3. Search using the IMSOC (Import Model) reference given in this service
  4. Within the relevant import notice in the search results, select 'Clone as' and the relevant CHED type:
    • CHED-P: Products of animal origin, germinal products and animal by-products
    • CHED-A: Live animals
