You can now submit a digital application to the EU and Northern Ireland

For applications to the EU and Northern Ireland, you can create a paperless EHC with data that matches the EU's legislative requirements (CHED) and their IT system (TRACES).

How it works

  1. You

    1. Create a digital application, similar to paper but some data you must select, such as commodity codes, and premises, such as cold stores. You can defer some questions to your certifier, such as transport details.

  2. Certifier

    2. The data is reviewed, digitally approved and validated against TRACES. The Certifier will no longer need to print and stamp amendments.

  3. Importer

    3. All data appears in TRACES as an EU Import Notice and is copied into a CHED, and the two are linked.

  4. Transporter

    4. You will give a QR code to the transporter enabling the BCP to locate the linked Import Notice and CHED.

What's remains the same

  • No requirement to print and stamp EHCs
  • Data validation with the EU’s requirements prior to dispatch
  • Data matching between the EU Import Notice and CHED
  • Reduced burden on importer when creating the CHED
  • Data can be changed after the goods have been dispatched in the digital service, such as transport changes or BCP feedback (Cancel & reissue)

What remains the same

  • Certifiers will continue to check data and the goods
  • Data (and sometimes goods) are checked by BCPs (a separate process to data validation)
  • A previous digital application can be copied

Before you start

Ensure you allow adequate time for your first application.

In order to use the data in your application, the person cloning the CHED will need to find the EU import notice. You must make the person that creates the CHED aware of this before you start. If you’re not sure who does this, contact your customer.
