We’re making some changes to the service

To make sure all export data is consistent with the EU’s trade and control expert system (TRACES), we’re making some changes to how you apply to move animals and animal products to the EU and Northern Ireland.

Later this year, you will be able to create a fully digital application for any consignment to the EU and Northern Ireland. With a fully digital application, your:

  • certifiers and importers will no longer need to use paper copies of EHCs
  • commodity data will be consistent with EU border control post requirements

To learn more, you can read about the changes to how you apply for an EHC.

You will still be able to use paper-based applications for consignments to Northern Ireland or the EU.
