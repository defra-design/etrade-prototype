We have changed the way you apply

To make sure all export data is consistent with the EU’s trade and control expert system (TRACES), we have made some changes to how you apply for an export health certificate (EHC).

If data is consistent between exporters in Great Britain and importers in Northern Ireland and the EU, exports are more likely to pass through border control posts and reach their destinations on time.

Certifiers and importers will no longer need to use paper copies of EHCs.

Before you submit your application to the certifier, you’ll need to enter or upload data for the export’s commodity group, weight and quantity. This data will be validated by TRACES before you submit the application to the certifier.

You can still use the previous EHC application service if you want to:

  • view an application that you started before 1 September 2022
  • apply for an EHC for exports to countries outside the EU or Northern Ireland
