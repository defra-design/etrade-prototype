Changes to the way you can apply for certificates to the EU

To make sure all export data is consistent with the EU’s trade and control expert system (TRACES), we’ve made some changes to how you apply for an export health certificate (EHC) to the EU and Northern Ireland.

You will be able to create digital certificates for the EU and Northern Ireland. If you apply for a digital EHC, your:

  • certifiers and importers will no longer need to use paper copies of EHCs
  • commodity data will be consistent with border control post requirements

To learn more, you can read about the changes to how you apply for an EHC.

You can use either service to apply for an EHC to Northern Ireland or the EU.
