You can now submit a digital application to send goods to the EU or Northern Ireland

To make sure all export data is consistent with the EU’s trade and control expert system (TRACES), we’re making some changes to how you apply to move animals and animal products to the EU and Northern Ireland.

You can create a paperless application for any consignment to the EU and Northern Ireland. With a paperless application, your:

  • certifiers and importers will no longer need to use paper copies of EHCs
  • commodity data will be consistent with EU border control post requirements

To learn more, you can read more about how digital applications work.
